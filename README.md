# 🔐 Password Strength Checker

## Overview

The Password Strength Checker is a Python-based cybersecurity tool developed as part of the DecodeLabs Cyber Security Internship Program.

The tool analyzes a user's password and evaluates its strength based on multiple security parameters. It helps users create stronger passwords by identifying weaknesses and providing improvement suggestions.

---

## Features

✅ Minimum Length Validation

✅ Uppercase Character Detection

✅ Lowercase Character Detection

✅ Numeric Character Detection

✅ Special Symbol Detection

✅ Common Password Blacklist Detection

✅ Password Strength Classification

✅ Security Improvement Suggestions

---

## Technologies Used

- Python 3
- String Module
- Conditional Statements
- Loops
- Cybersecurity Fundamentals

---

## Password Strength Criteria

The application evaluates:

1. Password Length
2. Uppercase Letters
3. Lowercase Letters
4. Numbers
5. Special Characters
6. Common Weak Password Patterns

Based on these checks, the password is classified as:

- Weak
- Medium
- Strong
